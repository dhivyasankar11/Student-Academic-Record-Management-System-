# Student Academic Record Management System

## Overview
This is a simple console-based program written in C to manage student academic records, including operations like adding, viewing, and searching records.

## Files
- student-academic-records.c: The main C source code file.

## How to Compile and Run
1.  *Save* the student-academic-records.c file locally.
2.  *Compile* the code using a C compiler (like GCC):
    bash
    gcc student-academic-records.c -o student_records
    
3.  *Run* the executable:
    bash
    ./student_records
